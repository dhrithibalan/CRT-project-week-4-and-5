Hangman Instructions
1. Run the game using python hangman.py
2. List of words used in the game is in wordbank.txt - each word is on a new line
3. When playing, user has 6 lives
4. When playing, user can enter one of the following
  - Single letters in upper or lower case
  - Multi letter strings only permitted for either "help" or "guess answer" (without the "")
    - "help" returns the characters guessed so far
    - "guess answer" allows user to guess the word, and directly leads to a win or a lose
5. Program continues to loop until user says "no" to replay
